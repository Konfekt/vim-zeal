This Vim plug-in sets `&keywordprg` (see `:help K`) so that hitting the key `K` shows the documentation of the keyword under the cursor in the offline documentation browser [Zeal](http://www.zealdocs.org).
Additionally, hitting the `<F1>` key shows a documentation entry page.

To look up the current keyword in many file types and configuration files in the terminal as well, have a look at [vim-keyword-lookup](https://github.com/Konfekt/vim-keyword-lookup).
